# Projeto Apolo

> O Apolo é um software de código proprietário e, por esse motivo, este repositório não disponibiliza seu código-fonte. Seu conteúdo se limita a uma documentação simplificada, com o objetivo de permitir que os colaboradores possam apresentar o Projeto Apolo em seus perfis e portfólios profissionais. <a href="projetoapolo.com"><strong>Acesse o Apolo clicando aqui</strong></a>.

## O que é o Apolo?
<p align="justify">
O <strong>Apolo</strong> é uma ferramenta gratuita e online desenvolvida para auxiliar no dimensionamento e orçamento de sistemas fotovoltaicos. A plataforma permite que qualquer pessoa, mesmo sem conhecimento técnico, gere de forma rápida e intuitiva uma estimativa completa de custo dos equipamentos necessários para a instalação de um sistema de energia solar, atendendo a diferentes perfis de consumo, como residencial, rural, comercial e industrial.
</p>

## Proposta
- Permitir que o próprio usuário realize simulações e orçamentos sem depender de intermediários;
- Reduzir burocracias comuns no mercado de energia solar;
- Preservar a privacidade do usuário, sem exigir cadastros ou armazenamento de dados pessoais;
- Disponibilizar a ferramenta gratuitamente e de forma totalmente online.

## Iniciativa
<p align="justify">
O <strong>Apolo</strong> é um projeto desenvolvido no âmbito do Colégio Técnico Industrial de Santa Maria (CTISM - UFSM), com o objetivo de democratizar o acesso a informações sobre energia solar e incentivar o uso de fontes renováveis. A plataforma foi criada como uma iniciativa acadêmica e tecnológica, buscando unir conhecimento técnico e inovação para oferecer uma solução prática, confiável e acessível a toda a comunidade.
</p>
